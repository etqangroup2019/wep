تم تجهيز المشروع بالكامل، وتم بناء النسخة النهائية (Build) وتجهيز مستودع Git محلي.

لقد قمت بالخطوات التالية:

تعديل ملف التجاهل .gitignore لضمان عدم رفع الملفات المؤقتة (مثل node_modules).
تثبيت المكتبات اللازمة (npm install).
بناء الموقع للتأكد من خلوه من الأخطاء (npm run build).
إنشاء مستودع Git محلي وحفظ كافة الملفات فيه (Commit).
الخطوات المتبقية لرفعه على GitHub:
عليك الآن إنشاء مستودع جديد على موقع GitHub، ثم تنفيذ الأوامر التالية لربط المشروع ورفع الملفات:

اذهب إلى GitHub - New Repository.
قم بتسمية المستودع (مثلاً my-website).
لا تقم بتهيئة المستودع (لا تضف README أو gitignore)، فقط اضغط Create repository.
انسخ رابط المستودع (الذي ينتهي بـ .git، مثلاً https://github.com/username/repo-name.git).
نفذ الأوامر التالية في التيرمينال (استبدل الرابط برابط مستودعك):
bash
git remote add origin <رابط-المستودع-هنا>
git push -u origin main
بعد ذلك سيتم رفع المشروع بالكامل إلى GitHub.
