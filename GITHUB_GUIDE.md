# دليل رفع التحديثات على GitHub

هذا الدليل يشرح كيفية رفع التعديلات الجديدة على موقع GitHub بكل سهولة.

## الخطوة 1: حفظ التعديلات محلياً

بعد الانتهاء من التعديلات على الكود، افتح "Terminal" واكتب الأوامر التالية بالترتيب:

1. **إضافة الملفات المعدلة:**
   ```bash
   git add .
   ```
   *هذا الأمر يجمع كل الملفات التي قمت بتعديلها.*

2. **تسجيل التعديلات (Commit):**
   ```bash
   git commit -m "اكتب هنا وصف للتعديلات التي قمت بها"
   ```
   *مثال:* `git commit -m "تحديث صفحة الخدمات وإصلاح الألوان"`

## الخطوة 2: رفع التعديلات إلى GitHub

بعد حفظ التعديلات محلياً، استخدم الأمر التالي لرفعها إلى الموقع:

```bash
git push
```

## ملخص سريع للأوامر

في كل مرة تريد رفع تحديث:
```bash
git add .
git commit -m "وصف التعديل"
git push
```

---
**ملاحظة:** إذا طلب منك تسجيل الدخول، قم بإدخال اسم المستخدم وكلمة المرور الخاصة بحساب GitHub. في حال كنت تستخدم المصادقة الثنائية (2FA) أو كلمات المرور الحديثة، قد تحتاج لاستخدام "Personal Access Token" بدلاً من كلمة المرور.
